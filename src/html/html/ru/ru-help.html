Что тут происходит?
Это игра в которой можно делать что угодно, давайте начнем с начала
Ты управляешь персонажем при помощи команд
Немного раскажу о разметке
{a, b, c} => Аргументы
[a] => Не обязательный аргумент
a || b => Один из элементов
(Привет!) => Комментарий, писать не надо
a... => Аргумент можно повторять
Давайте расмотрим команды
Множество команд содержат подкоманды, их можно описать так
cmd {command {args}}
see
Команда для просмотра чего-либо
Пример: see enemy 2
  • location (Информация о текущей локации)
  • enemy [id || n] (Существа на локации)
  • health (Уровень твоей жизни)
  • nick {nick || id} (id на указанном nick и наоборот)
  • road (Дороги с локации)
set
Устанавливает значения
Пример: set nick 1234 abcde
  • nick {id, nick} (Установить вместо id nick)
Остальные
  • attack {n || id} (Атаковать существо)
  • bury {n || id} (Похоронить труп)
  • echo {text} (Выводит вам введенный текст, тестовая команда)
  • go {n || id}... (Идти по дороге на другую локацию)
На этом все, предлагаю вернутся в игру
