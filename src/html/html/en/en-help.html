What's going on here?
This is a game in which you can do anything, let's start from the beginning.
You control the character using commands
I'll tell you a little about the markup
{a, b, c} => Arguments
[a] => Optional argument
a || b => One of the elements
(Hello!) => Comment, no need to write
a... => The argument can be repeated
Let's take a look at the commands
Many commands contain subcommands, they can be described as follows
cmd {command {args}}
see
Command to view something
Example: see enemy 2
  • location (Information about the current location)
  • enemy [id || n] (Creatures in the location)
  • health (The level of your life)
  • nick {nick || id} (id on the specified nick and vice versa)
  • road (Roads from the location)
set
Sets values
Example: set nick 1234 abcde
  • nick {id, nick} (Set nick instead of id)
Other
  • attack {n || id} (Attack the creature)
  • bury {n || id} (Bury the corpse)
  • echo {text} (Prints you the text you entered, test command)
  • go {n || id}... (Move along the road to another location.)
